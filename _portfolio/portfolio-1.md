## Overview
This project involved downloading and calibrating Landsat-8 satellite imagery 
from USGS Earth Explorer for an area of interest, and performing spectral 
analysis comparing uncalibrated, TOA and BOA images.

## Tools Used
- ArcGIS Pro
- USGS Earth Explorer
- Landsat-8 OLI/TIRS imagery

## Key Tasks
- Downloaded Landsat Collection 2 Level-1 and Level-2 imagery
- Performed TOA reflectance calibration in ArcGIS Pro
- Generated and compared RGB plots for all three images
- Created spectral plots using sample point locations
- Generated NDVI index from calibrated BOA image
- Calculated water body area from NDVI results

## Key Learnings
- Difference between uncalibrated, TOA and BOA reflectance
- LaSRC atmospheric correction model
- Spectral signature analysis of land cover types
- NDVI interpretation for vegetation and water mapping
